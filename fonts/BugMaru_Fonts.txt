■ばぐまるゴシックとは？

ばぐ★ＮＡＧＡが 1996 年頃からコツコツ作っていた X680x0 用ビットマップフォントを、
Windows・macOS など他機種でも使えるようにしたものです。
全体的に形を丸く、プログラミングでも使えるよう 0(ゼロ)と O(オー)などが区別できるように作りました。
なお、漢字部分は平木敬太郎氏の「秋葉原 16」をベースに変更・追加しています。

収録フォントは以下です。

1.「ばぐまるゴシック」
  等幅フォントで、コーディングなどに使えます。

2.「ばぐまるＰゴシック」
  プロポーショナルフォント(可変幅フォント)で、出力ウィンドウの文字など、
  できるだけ詰めて表示したい部分に使えます。

3.「BugMaru UI」
  2.と同じプロポーショナルフォントですが、コンピュータの設定画面などで使用するために
  さらに一部の文字の幅を小さめに調整したものです。

いずれもビットマップのみのフォント(ピクセルフォント)ですので、画面表示用としてご使用ください。

■特徴

・画面表示用・等幅ビットマップフォント
・コーディングで使えるように、0 O (ゼロとオー)、1 I l | (イチとアイとエルと縦棒)、
  , . (カンマとドット)、: ; (コロンとセミコロン)などが区別しやすい
・フォントサイズは 16x16 ピクセル(半角文字は半分の幅)の他に、12×12、8×8 ピクセルの小さいサイズも収録
・文字のスタイルは標準・太字(ボールド)・斜体(イタリック)・太字斜体(ボールドイタリック)
・一般的な英数字記号・ひらがな・カタカナ・漢字の他に、簡体字中国語の一部も収録
・プロポーショナルフォント(2 種類)も収録
・商用利用可能(条件あり)

■収録ファイル

このアーカイブファイルに収録されているフォントを以下に説明します。

● ばぐまるゴシック、ばぐまるＰゴシック、BugMaru UI (全フォント)
  BugMaruFonts.ttc

上記は、複数のフォント・スタイル・サイズが 1 つのパッケージになっている
True Type Collection (ttc) という形式のファイルです。
このファイルのみインストールすれば収録されているすべてのフォントが使用できます。

ただし、全フォント入りパッケージファイルはファイルサイズが大きいため、
Windows 7 以前の OS ではインストールできません。
過去のバージョンの OS で使用する場合は、以下のフォント別のファイルを使用してください。

● ばぐまるゴシック
  BugMaruGothic.ttc

● ばぐまるＰゴシック
  BugMaruP.ttc

● BugMaru UI
  BugMaruUI.ttc

上記 3 つは、各フォントの複数のスタイル・サイズが 1 つのパッケージになっている
True Type Collection (ttc) 形式のファイルです。

本ファイルのほか、特定のスタイル・サイズを使用したい場合に、
さらに分割した ttf 形式のファイルを別のアーカイブファイルとして配布しています。

以下の説明では、これらをまとめて「当フォント」と呼びます。

■フォントサイズ・スタイル

次のフォントが収録されています。

●フォント
・ばぐまるゴシック   … 等幅フォント
・ばぐまるＰゴシック … プロポーショナルフォント(可変幅フォント)
・BugMaru UI         … プロポーショナルフォント(可変幅フォント・UI 用)

●サイズ
・縦 16 ピクセル (半角 8×16 ピクセル・全角 16×16 ピクセル)
・縦 12 ピクセル (半角 6×12 ピクセル・全角 12×12 ピクセル)
・縦  8 ピクセル (半角 4× 8 ピクセル・全角  8× 8 ピクセル)

  ※「ピクセル」は「ドット」ともいいます。

●スタイル
・regular    … 標準
・bold       … 太字
・italic     … 斜体
・bolditalic … 太字斜体

■表示のされ方

●ビットマップ表示

Windows では、画面に表示する際に次のサイズを指定すると、ビットマップ形式での表示となります。

・12 ポイント
・ 9 ポイント
・ 6 ポイント
※ディスプレイの設定の「拡大縮小とレイアウト」が 100% の場合(96dpi)の場合。

これ以外のサイズを指定した場合や macOS では、ピクセルを模したアウトラインでの表示となります。

●アウトライン表示

当フォントに収録されているビットマップフォントは前述のサイズ・スタイルのみのため、それ以外の
サイズ・スタイルでまったく表示されなくなることを避けるためにアウトラインデータも収録しています。

アウトラインデータはピクセルの輪郭をアウトライン化したもので、拡大するとそれぞれの
ピクセルが大きくなります(カクカクした表示となります)。

環境によってはにじんだ表示になる等、きれいに表示されない場合がありますのでご注意ください。

■文字コードと収録文字

文字コードは Unicode で作成していますが、Unicode に含まれるすべての文字が収録されているわけでは
ありません。
収録されていない文字は、動作環境やアプリケーションによって表示が異なり、特定の文字("?" や "・"等)
が表示されるか、何も表示されないもしくは別のフォントの文字が表示されます。

縦 12 ピクセル・縦 8 ピクセルのフォントのデザインは、漢字や一部の文字は読みにくくなってしまうため
文字の形を丸くしていません。また一部は作成途中の文字があり整えていない文字(潰れた文字)があります。
ご了承ください。

収録されていない文字や作成途中の文字は、今後の更新で追加・修正していきたいと思いますが、
すぐに反映できるとは限りませんので、あまり期待はしないでください。

半角文字と全角ひらがな・カタカナ・記号はばぐ★ＮＡＧＡが作成していますが、
漢字は平木敬太郎氏の「秋葉原 16」 X680x0 版をベースに、一部の文字の修正および
JIS 第三水準漢字・JIS 第四水準漢字・中国語簡体字・繁体字の一部の文字の追加をしています。

■使用方法

●インストール

すでに同じフォント名のものがフォントフォルダにインストールされている場合は一度削除しないと
インストールできない場合がありますのでご注意ください。

アーカイブファイル(圧縮ファイル)を任意のフォルダに展開した上で、格納されているファイルに
対して以下の操作でインストールしてください。

【Windows の場合】

拡張子 ttc のファイルを Windows のフォントフォルダ(コントロールパネル内の「フォント」フォルダ、
または「Fonts」フォルダ)にドラッグ＆ドロップすることでインストールできます。

Windows Vista の場合、展開するフォルダの場所によってはインストールできないことがあります。
その場合は一度デスクトップや C ドライブ(システムドライブ)にコピーし、そこからフォントフォルダに
ドラッグ＆ドロップしてみてください。

他の方法として、展開したフォントファイルを選択し、右クリックメニューから「インストール」を
選択してもインストールすることができます。

【macOS の場合】

拡張子 ttf のファイルをダブルクリックすると、Font Book アプリケーションでプレビュー表示されます。
その画面で「インストール」を選択するとインストールされます。

他の方法として、ライブラリ(Library)フォルダ内の Fonts フォルダにドラッグ＆ドロップすることでも
インストールできます。

●アンインストール

実行しているすべてのアプリケーションが当フォントを使用していないことを確認してから、各 OS の
フォントフォルダから当フォントを削除してください。

■ばぐまるＰゴシックと BugMaru UI の違い

ばぐまるＰゴシックと BugMaru UI は可変幅フォントですが、以下の違いがあります。

・ばぐまるＰゴシック
  ばぐまるゴシックをベースに、半角英数記号、全角英数・ひらがな・カタカナ・一部の記号について、
  文字の余白を詰めたものです。

・BugMaru UI
  主にコンピュータの設定画面(ダイアログ)に使用できるように調整したフォントです。
  ばぐまるＰゴシックをベースに、全角英数・ひらがな・カタカナ・一部の記号の文字を編集して幅を
  狭くしています。また、半角の A～Z・a～z の幅を調整しています。これは、Windows では
  半角 A～Z・a～z の幅の平均値でダイアログのサイズを決めているためです。
  そのため、半角 M や W などの幅がばぐまるＰゴシックに比べて少し広くなっています。

■よくある質問と回答

●一部の文字が半角の幅になる
  一部のアプリケーションで、一部の記号が半角の幅として表示されることがあります。
  その場合、フォント自体が持っている幅を使用する設定をすることで解消される場合があります。

  例えば秀丸エディタでは、[その他]→[ファイルタイプ別の設定]→[フォント]を選択するとき、
  「プロポーショナルフォント扱い」にチェックを入れると、フォント自体が持っている幅を使用して
  表示されるようになります。

●文字がにじんだような表示になる
  以下の原因により、アウトラインデータを使用した表示になり、OS のアンチエイリアス処理により
  にじんだような表示になる場合があります。

  ・Windows で 6・9・12 ポイント以外のサイズを使用している
  ・ディスプレイの拡大・縮小の設定をしている(100% 以外の設定にしている)
  ・ビットマップフォント表示未対応のアプリを使用している
  ・macOS を使用している

  Windows で 6・9・12 ポイントのサイズを指定して、かつディスプレイの拡大・縮小設定を 100% に
  すると解消される場合があります。

  フォントのビットマップ部分の表示に対応していないアプリや、macOS ではアウトライン表示のみ
  対応しているため、にじみが解消できない場合があります。

● BugMaru UI Gothic が正しく表示されなくなった
  Ver.0.20 より、「BugMaru UI Gothic」のフォント名を「BugMaru UI」に変更しています。
  お手数ですが、該当フォントを使用していたアプリケーションのフォント設定を変更してください。

● Windows 7 以前の OS でスタイルが反映されない
  原因は不明ですが、スタイルが正しく反映されないケースがあるようです。
  フォント・サイズ・スタイルを個別のファイルにした ttf 版を配布していますので、
  そちらをご利用ください。
